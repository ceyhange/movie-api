## Description
Movie API app with Spring boot framework.

## Requirements

```bash
$ install postgres sql
$ install java 11
$ install maven 
```

## Running the app

```bash
# Start Postgres:
$  pg_ctl -D start
$  psql -U <user>

# development API URL:
$ http://localhost:8080/api/

# Swagger Document:
$ http://localhost:8080/swagger-ui/
```
